Fonti, licenze e limitazioni d’uso
Open-Meteo: dati forniti tramite API sotto licenza CC BY 4.0. È richiesta attribuzione e link alla licenza. Riferimenti: licenza · termini.
Radar DPC (Radar-DPC): visualizzazione tramite servizio WMS conforme a standard OGC. È una vista informativa, non un sistema di allerta. Riferimenti: servizi WMS/WMTS · pagina radar.
Nota tecnica: pagina statica. Se la apri in locale con file://, il fetch di bollettino.html può essere bloccato dal browser. Alcuni browser bloccano anche iframe di terze parti.
Note operative su lampi, Meteologix e responsabilità
Lampi (Blitzortung / LightningMaps): questa vista incorpora la mappa live via iframe. È un progetto volontario, non ufficiale. Uso informativo. Non usarla per sistemi di allerta, valutazioni di rischio o impieghi commerciali senza permessi espliciti. Riferimenti: termini · parametri embed.
Meteologix: offre prodotti professionali, inclusi radar e fulminazioni, con contenuti soggetti a licenza. Evitare embedding, mirroring o scraping senza accordo o permesso. Riferimenti: API · Satellite & Lightning · termini portale.
Responsabilità: la dashboard serve per situational awareness e consultazione operativa. Per allerta, danni o responsabilità servono fonti ufficiali e verifiche dedicate.
La pagina mantiene tutte le funzioni del file di partenza, ma con una struttura più leggibile e più professionale: controlli in alto, snapshot e trend al centro, strumenti live sotto, bollettino in chiusura.
